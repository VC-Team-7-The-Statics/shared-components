# Shared Components

바닐라 코딩 부트캠프의 7팀 The Statics의 팀 프로젝트에 사용되는 공통 컴포넌트 repository입니다.

## 사용 방법
### Install it
`npm install @the-statics/shared-components`

### import it
`import { Button } from '@the-statics/shared-components';`

### use it
`<Button text={"yolo"}></Button>`

### example
```
import { Button } from '@the-statics/shared-components';
import React from 'react';

function App() {
  return (
    <div>
      <Button text={"yolo"}></Button>
    </div>
  );
}

export default App;
```
